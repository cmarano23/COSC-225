Animated Dijkstra's Algorithm

Below, you can find an interactive and animated display of Dijkstra's Algorithm. You can click anywhere in the box to create vertices. Each vertex is labeled with a number, and you can click on a vertex to select it. Once a vertex is selected, you can select any other vertex to create an edge between the two vertices. The edge will be labeled with a weight, which you can change by entering in a number in the box next to "Enter Edge Weight" and then pressing the "Submit" button. The default value for the edge weight is 1. In order to begin the algorithm, you must select click a vertex and then press the button "Start". After this, you can either press the "Step" button to walk through the algorithm one step at a time, or you can press the "Animate" button to watch the algorithm run automatically.
